BLL - Business logic layer

Библиотека классов выполняет начальную обрботку запросов.
Каждый класс бизнес слоя может производить вызов методов из библиотеки Service.

Для обращения к библиотеке используется механизм управления зависимостями dependency injection. Чтобы получить доступ к классу, 
необходимо получить компонент ServiceProvider и получить класс методом GetService<InterfaceName>
