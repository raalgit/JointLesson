BLL - Business logic layer

Библиотека классов выполняет начальную обрботку запросов.
Каждый класс бизнес слоя может производить вызов методов из библиотек Utility и Service в зависимости от типа запроса.

Для обращения к библиотекам используется механизм управления зависимостями dependency injection. Чтобы получить доступ к классу, 
необходимо получить компонент ServiceProvider и получить класс методом GetService<InterfaceName>
